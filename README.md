pdftext
=======

## Basis
* Convert PDF to text Command Line Tool
* OS X

## How to convert

```objective-c
NSString *path = @"/path/to/pdf";
PDFDocument *document = [[PDFDocument alloc] initWithURL:[NSURL fileURLWithPath:path]];
NSString *string = document.string;
NSLog(@"%@", string);
```
